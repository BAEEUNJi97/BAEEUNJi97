주어진 실습 시간 동안 아래 이벤트에 대해서 제이쿼리 선택자 예제를 적용하여 이벤트 핸들러를 구현해 보세요. 다양한 조합에 대해서 테스트 해보세요 - 마우스 이벤트: "click": 클릭했을 때 발생하는 이벤트. 주로 버튼 클릭 등에 사용. "dblclick": 더블 클릭했을 때 발생하는 이벤트. "mouseover": 요소에 마우스를 올렸을 때 발생하는 이벤트. "mouseout": 요소에서 마우스가 벗어났을 때 발생하는 이벤트. "mousemove": 마우스가 요소 위에서 움직일 때 발생하는 이벤트. "mousedown": 마우스 버튼을 눌렀을 때 발생하는 이벤트. "mouseup": 마우스 버튼을 뗐을 때 발생하는 이벤트. - 키보드 이벤트: "keydown": 키를 눌렀을 때 발생하는 이벤트. "keyup": 키를 뗐을 때 발생하는 이벤트. "keypress": 키를 눌렀다 뗄 때 발생하는 이벤트. - 포커스 이벤트: "focus": 요소가 포커스를 받았을 때 발생하는 이벤트. "blur": 요소가 포커스를 잃었을 때 발생하는 이벤트. - 폼 이벤트: "change": 입력 값이 변경되었을 때 발생하는 이벤트. 주로 폼 요소에서 사용. "submit": 폼을 제출할 때 발생하는 이벤트. - 창 이벤트: "resize": 창 크기가 변경되었을 때 발생하는 이벤트. "scroll": 페이지를 스크롤할 때 발생하는 이벤트. - 기타: "load": 페이지 또는 이미지 등이 로드되었을 때 발생하는 이벤트. "unload": 페이지가 언로드(닫힘)되었을 때 발생하는 이벤트.
